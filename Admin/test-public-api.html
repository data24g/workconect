🧪 Test Public Article API

Test các endpoint public để web client có thể lấy bài viết đã publish.

1. GET /api/articles - Tất cả bài viết đã publish

2. GET /api/articles/menu/{menu} - Bài viết theo chuyên mục

3. GET /api/articles/latest?limit={n} - Bài viết mới nhất

4. GET /api/articles/search?q={keyword} - Tìm kiếm bài viết

5. GET /api/articles/{id} - Chi tiết bài viết
